# **App Name**: Assigno

## Core Features:

- Firebase Auth Portal: Secure student onboarding and session management using email/password authentication.
- Assignment Command Center: Full CRUD management for assignments, persisting data directly to Firebase Firestore including titles, subjects, and priorities.
- Live Performance Dashboard: Real-time summary tiles showing Total, Completed, Pending, and Overdue assignments with a visual progress percentage tracker.
- Deadline Pulse: Automatic 24-hour proximity alerts and distinct visual flagging for overdue items to ensure academic compliance.
- AI Study Breakdown Tool: Leverages generative AI to analyze an assignment's description and generate a suggested tool-driven action plan of sub-tasks.
- Chrono-Filter & Search: A high-performance search and filter interface to instantly sort assignments by subject, priority, or completion status.
- Integrated Calendar Sync: A streamlined visual calendar interface for a chronological overview of all pending academic deadlines.

## Style Guidelines:

- Primary color: Deep Acadia Blue (#394BAC), chosen for its association with focus and reliability in educational settings.
- Background color: Mist Gray (#F6F6F9), a desaturated variation of the primary hue to provide a clean, high-contrast canvas.
- Accent color: Celestial Azure (#3CB4DD), an analogous hue that provides high contrast for call-to-action buttons and notifications.
- Font pairing: 'Space Grotesk' for modern, tech-forward headlines and 'Inter' for highly legible, objective body and data text.
- Minimalist line-based icons for academic categories (book, pen, clock) to maintain a clutter-free interface.
- Modular card-based dashboard layout with distinct status-colored borders (red for overdue, green for completed) and fluid responsive grid transitions.
- Soft CSS transitions for task completion status changes and subtle entry fades for dashboard cards during page load.
